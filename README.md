# Racha

## Setup 

### Intialize MongoDB database
```
$ cd racha
$ mkdir data
$ cd data
$ mkdir db
$ mongod --dbpath db
```

### Download dependencies
```
$ npm install
$ bower install
```

### Build and run server
```
$ grunt build
$ grunt serve
--> Default: locahost:9000
```
